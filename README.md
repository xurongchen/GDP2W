GDP2W
===========================
GDP2W 是毕业设计材料PDF转Word（ Graduation Design materials Pdf2Word ）的缩写。

本项目的目的是方便毕设过程中按照PDF给定格式的材料撰写。

项目文件夹包括原有PDF和转换后的Word两部分，转换工作是在Word2016版中进行的。
